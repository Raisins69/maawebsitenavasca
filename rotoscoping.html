The Rotoscoping Process

1

Video Reference Preparation

The first step involves selecting and preparing the video reference. This includes choosing the right footage and adjusting the frame rate for optimal rotoscoping results.

2

Download FlipaClip

Download and install FlipaClip — a powerful frame-by-frame animation app that serves as the primary tool for creating the rotoscope animation.

3

Create a New Project

Create a new project in FlipaClip by selecting the desired canvas size and setting the frames per second (FPS) to match the intended animation speed.

4

Import the Video Reference

Tap the three-dot menu within FlipaClip and select "Add a Video" to import the reference footage. This allows you to trace over each frame directly.

5

Rotoscope Each Frame

Carefully trace over each frame of the video reference to create the rotoscoped animation. This is the most time-intensive step, requiring precision and consistency across every frame.

6

Final Edits & Export

Apply finishing touches by adding the original video side-by-side for comparison, incorporating personal branding elements, and exporting the completed animation.
